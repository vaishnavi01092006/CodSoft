# Image Captioning AI
## Description
This project is an AI-based Image Captioning System developed using Python. It generates a simple caption for an image by recognizing the image category using predefined rules.
## Features
- Image caption generation
- Rule-based image recognition
- Beginner-friendly Python project
- Simple and easy to understand
- Console-based application
## Technologies Used
- Python
- Artificial Intelligence
- Rule-Based System
## How to Run
1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the following command:
```bash
python image_captioning.py
